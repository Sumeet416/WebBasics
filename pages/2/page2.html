Html Fundamentals

Understanding the structure, syntax, and essential elements of HTML

WT
WebBasics Team Mar 22, 2025 • 4 min read

HTML is the foundation of every webpage. Learn about tags, attributes, document structure, and the most commonly used elements to start building your own sites.

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a webpage using a system of elements represented by tags. Browsers interpret HTML to render content like headings, paragraphs, links, images, and more.

HTML is not a programming language; it’s a markup language that gives meaning and structure to content. Every website you visit relies on HTML at its core.

HTML Document Structure

An HTML document has a specific skeleton that every browser expects. Here’s the minimal structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
  </head>
  <body>
    <!-- Content goes here -->
  </body>
</html>
  • <!DOCTYPE html> – Declares the document type and HTML version.
  • <html> – The root element that wraps all content.
  • <head> – Contains meta information, title, links to CSS, and scripts.
  • <body> – Holds the visible content of the webpage.

Tags, Elements, and Attributes

Tags are the building blocks. Most come in pairs: an opening tag <tagname> and a closing tag </tagname>. The content between them, plus the tags themselves, form an element.

Attributes provide additional information about an element. They appear inside the opening tag:

<a href="https://example.com">Click here</a>

Here href is an attribute that specifies the link destination.

Common global attributes include id, class, and style.

Essential HTML Elements

Here are some fundamental elements you’ll use constantly:

  • Headings – <h1> to <h6> for section titles, with <h1> being the most important.
  • Paragraphs – <p> for blocks of text.
  • Links – <a> with href to create hyperlinks.
  • Images – <img> with src and alt attributes.
  • Lists – <ul> (unordered), <ol> (ordered), and <li> for list items.
  • Divisions and Spans – <div> (block container) and <span> (inline container) for grouping content.

Example of a simple page using these elements:

<h1>My First Page</h1>
<p>Welcome to my website.</p>
<a href="about.html">Learn more about me</a>
<img src="photo.jpg" alt="My photo">
<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

Nesting and Hierarchy

HTML elements can be nested inside each other to create complex structures. Proper nesting is crucial for both styling and accessibility.

<div>
  <h2>About Us</h2>
  <p>We are a team of <strong>web developers</strong>.</p>
</div>

Always close tags in the reverse order they were opened. This creates a tree‑like structure known as the Document Object Model (DOM).

Comments

Comments help you document your code. They are not displayed in the browser:

<!-- This is a comment -->

Use comments to explain sections, leave notes, or temporarily disable code.

Next Steps

Now that you know the basics of HTML syntax and essential elements, you’re ready to style your content with CSS. But before moving on, practice by building a simple webpage that includes headings, paragraphs, links, images, and lists. Experiment with nesting and attributes.
