Introduction to Web Build

Understanding the core languages, frameworks, and your first project setup

WT
WebBasics Team Mar 22, 2025 • 5 min read

New to web development? Learn the distinct roles of HTML, CSS, and JavaScript, how frameworks differ from languages, and get comfortable with the Atom editor as you build your first project.

The Three Pillars of the Web

Every website you visit is built on three fundamental technologies: HTML, CSS, and JavaScript. They work together to create structure, style, and interactivity. Think of them as the skeleton, skin, and muscles of a webpage.

  • HTML (HyperText Markup Language) – Provides the raw structure and content. It defines headings, paragraphs, images, links, and other elements that give meaning to the information.
  • CSS (Cascading Style Sheets) – Handles presentation: colors, fonts, spacing, layout, and responsive design. It makes the page visually appealing and adaptable to different screen sizes.
  • JavaScript – Adds behavior and interactivity. It can update content dynamically, respond to user actions, fetch data from servers, and create engaging experiences.

Together, these three languages form the foundation of front‑end web development. Understanding how they interact is essential before moving on to more advanced tools.

Languages vs. Frameworks: What’s the Difference?

As you start learning, you’ll hear about frameworks like React, Vue, Angular, or Bootstrap. It’s important to know the distinction between a language and a framework.

  • Languages (HTML, CSS, JavaScript) are the core building blocks. They are standards defined by specifications, and browsers implement them directly. You can write a website using only these languages.
  • Frameworks are pre‑written libraries or tools built on top of languages to speed up development. For example, Bootstrap provides ready‑made CSS classes and components, while React helps you build complex user interfaces with JavaScript. Frameworks are optional but often used in professional projects.

Think of languages as the ingredients and frameworks as recipes. You can cook without recipes, but they help you create consistent, scalable results faster. However, mastering the languages first gives you the flexibility to understand and debug any framework.

Setting Up Your First Project with Atom

To write HTML, CSS, and JavaScript, you need a text editor. Atom is a free, open‑source editor that’s beginner‑friendly yet powerful. Here’s how to set up a basic website project:

  1. Install Atom – Download from atom.io and install it on your computer.
  2. Create a project folder – On your desktop or documents, create a new folder called my-first-website.
  3. Open the folder in Atom – Launch Atom, then go to File → Add Project Folder and select your new folder. This becomes your project root.
  4. Create your first HTML file – Inside the folder, right‑click in the sidebar and choose New File. Name it index.html. This file will be the homepage.
  5. Add a CSS file – Create another file called style.css. Link it in your HTML with <link rel="stylesheet" href="style.css"> inside the <head>.
  6. Add a JavaScript file – Create script.js. Link it just before the closing </body> tag using <script src="script.js"></script>.

Your project structure should look like this:

my-first-website/
├── index.html
├── style.css
└── script.js

Atom provides syntax highlighting, autocompletion, and a built‑in package manager to extend functionality. You can preview your site by opening index.html in any web browser.

Writing Your First Lines of Code

Now that your environment is ready, let’s write a simple webpage that demonstrates all three languages.

index.html – The structure:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Website</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <h1>Hello, Web!</h1>
    <p id="message">This is a paragraph.</p>
    <button id="clickBtn">Click Me</button>
    <script src="script.js"></script>
  </body>
</html>

style.css – Add some style:

body {
  font-family: Arial, sans-serif;
  text-align: center;
  margin-top: 50px;
  background-color: #f4f4f4;
}

h1 {
  color: #333;
}

button {
  padding: 10px 20px;
  font-size: 16px;
  cursor: pointer;
}

script.js – Add interactivity:

const button = document.getElementById('clickBtn');
const message = document.getElementById('message');

button.addEventListener('click', () => {
  message.textContent = 'You clicked the button! 🎉';
  message.style.color = 'blue';
});

Save all files, then open index.html in your browser. You’ll see a styled page with a button that updates the paragraph when clicked. This simple example shows how HTML, CSS, and JavaScript work together.

Next Steps

Congratulations! You’ve built your first interactive webpage. From here, you can expand by learning:

  • More HTML tags (forms, tables, media)
  • CSS layout techniques (Flexbox, Grid)
  • JavaScript fundamentals (variables, functions, DOM manipulation)
  • How to use a framework like Bootstrap or React after mastering the basics

Remember, every expert was once a beginner. Keep experimenting, break things, and rebuild them. That’s how you grow as a developer.
